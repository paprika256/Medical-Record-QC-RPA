项目名称：基于 RPA 技术的病案首页智能质控助手
1. 项目背景
痛点：临床医生填写病案首页耗时长，且容易出现逻辑错误（如手术与麻醉不符），导致医院国考指标失分。
目标：开发一款轻量级、非侵入式的桌面工具，实现填报时的实时自动核查。
2. 核心功能 
自动化校验： 自动扫描病案首页关键字段。
逻辑纠错： 内置 20+ 条临床逻辑规则（如：抢救次数>0 必须对应危重标识）。
一键报告： 生成质控报告，高亮显示错误项。
3. 技术栈
语言：Python 3.9
GUI框架：Tkinter / PyQt (看你用的啥)
数据处理：Pandas
打包工具：PyInstaller
