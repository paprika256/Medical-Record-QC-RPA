# 🏥 病案首页智能质控助手
>
> 💡 **核心价值**：基于 RPA 技术与临床逻辑规则，解决病案首页填报耗时、易错痛点，实现非侵入式的自动化质控。

---

## 📖 项目背景

在三甲医院的日常工作中，病案首页的数据质量直接关系到 DRGs/DIP 付费及国考指标。然而，传统的人工核查方式存在以下痛点：
*   **重复劳动多**：医生需要花费大量时间核对基础信息。
*   **逻辑错误难查**：如“手术方式”与“麻醉方式”不匹配、“危重患者”漏填抢救次数等隐蔽逻辑错误，人工难以肉眼识别。
*   **反馈滞后**：往往在病案归档后才发现问题，修改流程繁琐。

本项目旨在开发一款**轻量级、桌面端**的辅助工具，在不改造医院现有 HIS 系统的前提下，通过 RPA (机器人流程自动化) 技术实现“伴随式”质控。

## ✨ 核心功能 

*   **⚡ 自动化数据提取 **：
    *   能够自动扫描并提取病案首页的关键字段（诊断编码、手术操作、费用明细等）。
*   **🛡️ 智能逻辑校验 **：
    *   内置 **20+ 条** 临床硬性规则，包括：
        *   **完整性校验**：身份证号位数、必填项检查。
        *   **医学逻辑校验**：手术级别与职称匹配、麻醉费用与麻醉方式匹配、输血量与输血费用匹配等。
*   **📊 一键生成报告**：
    *   校验完成后立即生成可视化报告，高亮显示错误字段，并给出修改建议。
